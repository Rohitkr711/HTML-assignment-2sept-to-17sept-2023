1. 'head-tag' = It Contains metadata about the document, such as title, charset, and links to external stylesheets and scripts.
  2. 'body-tag' = Body tag contains the main content of the document, including any HTML elements, text, and images.
  3. 'h1-tag' = h1 tag defines a level 1 header, typically used for sailing a main title or subheading.
  4. 'p-tag' = p tag defines a paragraph of text, often used for describing a concept or providing information.
  5. 'ul-tag' = ul tag defines a list of items, often used for providing a list of options or items for the user to choose from.
  6. 'ol-tag' = ol tag defines a list of items, often used for listing items in order wise.
