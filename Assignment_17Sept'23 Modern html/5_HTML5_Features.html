Audio & Video Tags

With the help of Audio and Video tags, developers can embbed videos and audios into their websites. For styling the video tag, developers can use CSS and CSS3.


Canvas Tag

The canvas tag in HTML5 helps users to draw graphics or images on the fly using JavaScript. We can use it for drawing paths, boxes, circles, adding images, etc. The canvas tag has two attributes: width and height for setting the width and height of the canvas.
