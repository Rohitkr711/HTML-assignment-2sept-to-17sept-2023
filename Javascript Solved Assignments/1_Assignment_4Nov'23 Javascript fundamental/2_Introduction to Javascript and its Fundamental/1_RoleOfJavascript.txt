Q) Explain what JavaScript is and its role in web development.
Ans-> JavaScript is a high-level programming language commonly used for web development. It is a dynamic, interpreted, and object-oriented
      language that allows developers to create interactive and engaging content for websites. JavaScript allows developers to manipulate
      the Document Object Model (DOM) of a webpage, enabling them to change the layout, style, and behavior of web pages in real-time. 
      It is also used for server-side programming with the Node.js runtime environment. In summary, JavaScript is an essential tool for
      modern web development, allowing developers to create dynamic and engaging user experiences.
